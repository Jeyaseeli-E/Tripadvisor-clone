TripAdvisor Clone 🌍

A responsive TripAdvisor-inspired travel website built using HTML5 and CSS3. This project recreates the modern TripAdvisor user interface with a focus on clean design, responsive layouts, and interactive user experience.

Features ✨
Responsive design for Mobile, Tablet, Laptop, and Desktop screens
Sticky navigation header
Search bar with modern UI design
Destination and travel category sections
Auto-sliding hero banner using CSS animations
Interactive hover effects on cards and buttons
Travel recommendation cards
Popular destinations section
Experience and activity listings
Tripadvisor Foundation donation banner
Awards and promotional sections
Professional footer with social media links
Built using Flexbox and CSS positioning techniques
Technologies Used 🛠️
HTML5
CSS3
Flexbox
CSS Animations
Media Queries
Font Awesome Icons
Google Fonts

Project Goal 🎯

The objective of this project was to recreate the visual design and user experience of the TripAdvisor website while improving front-end development skills using only HTML and CSS.
